There are currently effectively 2 onboarding choice screens in the account creation flow.

1) User creates an employee account
2) User chooses "Join a company"
3) User enters the employee/company invite code
4) User clicks the join/next button
5) Instead of finishing properly, the app sends them to another "How will you use EcoLogic?" choice screen

That second choice screen should NOT appear in the employee join flow.

Even worse:
- On that second choice screen, if the user clicks "Create a company"

- So the routing/state is inconsistent and broken
