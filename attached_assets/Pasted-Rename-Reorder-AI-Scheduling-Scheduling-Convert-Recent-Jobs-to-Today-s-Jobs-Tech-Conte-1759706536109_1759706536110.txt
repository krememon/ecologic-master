Part 2 — Home Card: “Recent Jobs” → “Today’s Jobs” (show scheduled jobs for a date)

Goals

Replace the Home widget labeled “Recent Jobs” with “Today’s Jobs”.

Default date = today (user’s local timezone).

Show all jobs scheduled for that date. If none, show an empty state.

Include a compact date picker to jump dates (inline or popover).

Data Assumptions (adjust to actual schema)

Each job has one of:

job.schedule.start and job.schedule.end as ISO strings, or

job.scheduledAt as ISO string.

Status is at job.status (e.g., "Active"). Title at job.title. Optional: job.clientName, job.address.

Filtering

Convert selected date to a day range in the user’s timezone:

startOfDay(localDate) → endOfDay(localDate)

Include jobs where any of:

schedule.start is within range, or

scheduledAt is within range, or

job spans the day (start < startOfDay && end > endOfDay).

Exclude canceled jobs by default unless product says otherwise.

UI/UX

Card header: Today’s Jobs + small date picker control (e.g., button that opens a calendar).

Each row: icon, Job Title, optional time window (e.g., 9:00–11:00), status pill, and (if room) a single-line address or client.

Click row → navigate to job detail: /jobs/:id.

Empty state: “No jobs scheduled for this date.”

Loading skeletons for 3–5 rows.

Implementation Steps

Component rename: RecentJobs.tsx → TodaysJobs.tsx (or create new and swap import in Home).

Hook: useTodaysJobs(date) that:

Accepts a Date (default new Date()).

Computes day start/end in local tz.

Calls existing data layer (e.g., /api/jobs?date=YYYY-MM-DD) or filters client-side if jobs already cached.

Returns { jobs, isLoading, error }.

Date picker:

Use shadcn Popover + Calendar if available; else a minimal input type="date".

Badge:

Reuse existing status badge styles.

Routing:

Ensure clicking a job opens job detail without breaking back navigation.

Pseudocode (condensed)

// src/components/home/TodaysJobs.tsx
export default function TodaysJobs() {
  const [date, setDate] = useState(new Date());
  const { start, end } = dayBoundsLocal(date); // startOfDay/endOfDay using local tz
  const { jobs, isLoading, error } = useJobsByRange(start, end); // implement or call API

  return (
    <Card>
      <CardHeader className="flex items-center justify-between">
        <div className="text-xl font-semibold">Today’s Jobs</div>
        <DatePicker value={date} onChange={setDate} />
      </CardHeader>
      <CardContent>
        {isLoading ? <SkeletonRows count={4} /> :
         error ? <ErrorState /> :
         jobs.length === 0 ? <EmptyState text="No jobs scheduled for this date." /> :
         <ul className="space-y-2">
           {jobs.map(j => (
             <li key={j.id}>
               <Link to={`/jobs/${j.id}`} className="flex items-center justify-between rounded-xl p-3 hover:bg-muted/60">
                 <div className="flex items-center gap-3">
                   <Building2 className="h-8 w-8 opacity-80" />
                   <div>
                     <div className="font-medium">{j.title}</div>
                     <div className="text-sm text-muted-foreground">
                       {formatWindow(j.schedule)}{j.clientName ? ` • ${j.clientName}` : ""}
                     </div>
                   </div>
                 </div>
                 <StatusBadge status={j.status} />
               </Link>
             </li>
           ))}
         </ul>}
      </CardContent>
    </Card>
  );
}
