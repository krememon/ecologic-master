Adjust EcoLogic Homepage Layout – Move Status Tabs Above “Today” Section

CURRENT
The homepage shows the “Today / Jobs Scheduled Today” section
before the horizontal status tabs (Jobs Today, Open Jobs, etc.).

GOAL
Reorder the homepage so that the horizontal status tabs
appear ABOVE the “Today” (jobs scheduled today) section.

This provides quick business context first,
followed immediately by the user’s actual work for the day.

DO NOT redesign components.
This is a layout order change only.
