- Task code (optional)     -> price_book_items.task_code (NEW column)
- Service price (required) -> price_book_items.price

UI style:

DB changes (required)

Update API:
- GET /api/price-book-items returns these fields
- POST /api/price-book-items accepts and stores these fields
