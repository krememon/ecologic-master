- Today
- Upcoming

Example:
- I have an estimate created for today
- It is not appearing under Today or Upcoming on Home

This also needs to work for employees:
- If an estimate is assigned to an employee, it should appear on that employee’s Home tab as well

GOAL

1) owner/admin/company side
2) employee side when assigned

EXPECTED BEHAVIOR

OWNER / OFFICE SIDE
- If an estimate is scheduled for today, it should appear under Today on Home
- If an estimate is upcoming, it should appear under Upcoming on Home

EMPLOYEE SIDE

- If assigned and upcoming, it should appear under Upcoming on Home
