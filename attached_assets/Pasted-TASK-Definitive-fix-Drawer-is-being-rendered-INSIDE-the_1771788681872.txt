WHY PREVIOUS CSS DIDN’T WORK
If the drawer is mounted inside a container that already has transforms/positioning (common in wrappers), top/height CSS won’t move it correctly. The reliable fix is to render the drawer in a portal attached to document.body (or a dedicated #overlay-root), then safe-area works.
