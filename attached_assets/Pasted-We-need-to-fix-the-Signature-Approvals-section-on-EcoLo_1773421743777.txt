We need to fix the Signature Approvals section on EcoLogic Jobs so collected signatures actually appear there.

ISSUE
Right now signatures are being collected through the flows, but they are not properly showing in the Job → E-signature Approvals tab / Signature Approvals area.

Expected behavior:
- when an estimate is approved and signed, that signature should appear in Signature Approvals
- when a payment is collected and signed, that signature should also appear in Signature Approvals

So both types of signatures need to surface in the same approvals/history area for that job.

CURRENT SYMPTOMS
- signature collection modal appears
- user can draw/save signature
- but the Signature Approvals section still looks empty / incomplete
- “Payment Completion Signatures” and/or “Signature Requests” are not reflecting the real saved signatures

GOAL
Make all collected job-related signatures show correctly in the Signature Approvals tab for the job.
