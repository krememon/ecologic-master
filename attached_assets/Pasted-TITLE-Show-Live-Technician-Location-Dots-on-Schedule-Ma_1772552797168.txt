TITLE: Show Live Technician Location Dots on Schedule Map (Tech + Owner)

CONTEXT:
Geo tracking is now working on iOS: Capacitor Geolocation plugin returns coords, and the 1-point batch send is accepted (batch OK accepted=1). However, ScheduleMapView shows no dot because it only renders scheduled mapItems (currently 0). We removed viewer GPS UI earlier, so we now need a dedicated “Live Location” layer.

GOAL:
1) In TECHNICIAN map view, show “My Location” dot when clocked in (using latest tracked point).
2) In OWNER/ADMIN map view, show live locations for all currently clocked-in techs (poll /api/location/live).
3) Keep the existing schedule map pins unchanged; live layer should be additive.

REQUIREMENTS:
- Do NOT re-add “Locate me” UI or viewer GPS banners.
- “Hidden not disabled” permissions: only Owner/Admin (and maybe Dispatcher/Supervisor if they can see crew) can see other techs.
- Tech can only see their own dot.
- Map should not crash if no points exist.
- Add lightweight logs only during implementation, remove noisy ones after confirmed.
