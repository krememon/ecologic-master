- Behavior must be idempotent and safe (no accidental data loss).
- Preserve ability to unarchive job (owner/admin UI to restore).

BACKEND: new helper & wiring (Express / Node)

A) Helper: recomputeAndMaybeArchiveJob(jobId, actorUserId)
- Purpose: check whether job is fully paid (all invoices for job fully paid). If yes and job.is_archived = false then archive it.
- Steps:
  1. Query invoices for jobId:
     SELECT id, total_cents
     FROM invoices
     WHERE job_id = $jobId;
  2. For each invoice compute paid_cents:
     SELECT COALESCE(SUM(amount_cents),0) AS paid_cents
     FROM payments
     WHERE invoice_id = $invoiceId
       AND LOWER(status) IN ('succeeded','paid','completed');
  3. If every invoice.paid_cents >= invoice.total_cents then mark job archived:
     UPDATE jobs
     SET is_archived = true, archived_at = now(), archived_by = $actorUserId
     WHERE id = $jobId AND is_archived = false
     RETURNING id;
  4. Return boolean archived (true if archived).

- Use a DB transaction or at least consistent read for sums.

B) Where to call it:
- Primary: After webhook finalizes a payment (payment_intent.succeeded flow)
  In webhook handler after:
    - setting payment.status='succeeded';
    - recomputing invoice totals and invoice.status;
    - trigger recomputeAndMaybeArchiveJob(invoice.job_id, webhookActorUserId)
  (actorUserId can be SYSTEM_UUID or webhook service account id)
- Secondary: After manual payment route (cash/check) finalizes:
  - call recomputeAndMaybeArchiveJob(invoice.job_id, req.user.id)
- Tertiary: After any admin action that marks invoice paid (if exists), call helper too.

C) Idempotency:
- Helper must be idempotent — if already archived, no-op.
- Use `WHERE is_archived = false` on UPDATE to avoid toggling archived_at repeatedly.

D) Audit/logging:

SAMPLE PSEUDOCODE (Node/Express + pg client)
async function recomputeAndMaybeArchiveJob(jobId, actorUserId) {
  const invoices = await db.query('SELECT id, total_cents FROM invoices WHERE job_id=$1', [jobId])
  if (invoices.length === 0) return false
  for (const inv of invoices) {
    const { rows } = await db.query(
      `SELECT COALESCE(SUM(amount_cents),0) AS paid_cents
       FROM payments WHERE invoice_id=$1 AND LOWER(status) IN ('succeeded','paid','completed')`, [inv.id])
    inv.paid_cents = +rows[0].paid_cents
    if (inv.paid_cents < inv.total_cents) return false
  }
  // all invoices fully paid, archive
  const res = await db.query(
    `UPDATE jobs SET is_archived=true, archived_at=now(), archived_by=$1
     WHERE id=$2 AND is_archived=false RETURNING id`, [actorUserId || SYSTEM_USER_UUID, jobId])
  if (res.rowCount > 0) {
    logger.info('[archive] job archived', { jobId, archivedBy: actorUserId })
    return true
  }
  return false
}
