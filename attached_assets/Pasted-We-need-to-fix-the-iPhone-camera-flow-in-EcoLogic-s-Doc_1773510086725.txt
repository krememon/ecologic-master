ISSUE

the app crashes.

1. tap “Take Photo or Video”
2. open the actual iPhone camera

6. upload it normally into the selected folder

- user taps file picker area
- action sheet appears:
  - Photo Library
  - Take Photo or Video
  - Choose File
- if user taps Take Photo or Video:
  - native camera opens successfully
  - no crash
  - after capture, file/image is returned to the modal

Fix the actual native camera integration, not just web fallback.
