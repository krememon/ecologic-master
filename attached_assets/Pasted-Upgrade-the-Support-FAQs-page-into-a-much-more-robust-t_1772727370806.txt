Upgrade the Support → FAQs page into a much more robust troubleshooting/help center. Right now it’s too basic. I want it to be very detailed, include a lot more questions, and have a search bar so users can quickly find answers.

Goal

Turn the FAQ page into a real self-serve troubleshooting page that helps users solve common issues without needing to contact support.

Core requirements
1) Add a search bar

At the top of the FAQ page, add a clean search bar that lets users search FAQ questions/answers.

Search should:

filter results live as the user types

match against both question titles and answer content

be fast and simple

show a clean “no results found” state if nothing matches

Keep the search bar styled to match EcoLogic’s vibe.

2) Add a lot more FAQ content

Expand the FAQ page significantly. I want many more questions and much more detailed answers, especially focused on troubleshooting.

Organize FAQs into useful sections like:

Getting Started

Account & Login

Clock In / Time Tracking

Schedule

Jobs

Notifications

Documents

Estimates & Invoices

Payments & Billing

Mobile App Troubleshooting

Permissions / Access

Support & Reporting Issues

You can use section headers, grouped accordions, or another clean structure — but keep it mobile-friendly and easy to scan.

3) Make the answers detailed and practical

The FAQ answers should not be one-line filler.

They should be:

clear

actually helpful

troubleshooting-focused

step-by-step when appropriate

written in plain English

For troubleshooting questions, answers should explain:

what might be causing the issue

what the user should check first

what steps to try next

when to contact support if it still isn’t resolved
