- User opens EcoLogic on Samsung

- Live tracking does not clearly begin

Goal:

Audit and fix this exact flow:

1. Find the Android clock-in path in:
- Home.tsx
- geoTracking.ts
- androidLocationTracking.ts
- any timesheet/clock-in action helpers

2. Confirm the sequence on Android is exactly:
A. user clocks in
B. call checkAndroidPermissions()

- start() is called before requestForegroundPermission()
