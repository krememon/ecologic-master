- Must work reliably even if webhook finishes slightly after the return page loads.
- Must not double-open.

- Recovery: if user refreshes, must still be able to sign via pending localStorage or banner.

File: route handler for GET /stripe/return (logs show “[StripeReturn] Server-side hit: /stripe/return”)
- Parse query params sent by Stripe (Stripe will send `session_id` if we include it in success_url).

- success_url = `${baseUrl}/stripe/return?session_id={CHECKOUT_SESSION_ID}`
