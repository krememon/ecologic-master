- not just “disable payouts”

This is for internal/dev/admin use only in Dev Tools.

- Disconnect Stripe Account

4. They must re-enter all required Stripe onboarding information again
5. Any UI that checks Stripe-connected state should update accordingly

IMPORTANT

- Do NOT redesign Dev Tools more than needed
